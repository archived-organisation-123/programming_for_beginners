A Beginners Intro to Coding

You will need to download processing from processing.org in order to take part in this workshop.

Objectives

By the end of this workshop you should be able to:

  • Name some different data types
  • Understand the basic structure of a program
  • Draw some graphics to screen
  • Capture keyboard input
  • Love Programming!

Processing

  • Download processing from processing.org
  • Language and platform for teaching programming in a visual context
  • Simple to get up and running, but extremely powerful!

Some Important Data Types

  • String (a bit of text)
    "Hello, world!"
  • int (a whole number or integer)
    101
  • float (a decimal number)
    0.25
  • boolean (true or false)
    true
  • object (a complex type)
  • Array (a list of data)

Conditional Statements

							
								void keyPressed() {
									if(keyCode == 37){
										println("LEFT key pressed");
									} else if(keyCode == 38){
										println("UP key pressed");
									} else if(keyCode == 39) {
										println("RIGHT key pressed");
									} else if(keyCode == 40) {
										println("DOWN key pressed");
									}
								}
